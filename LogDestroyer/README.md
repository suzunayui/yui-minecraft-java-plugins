# LogDestroyer

木の斧で木を一括破壊するMinecraftプラグインです。

## 機能

- 木の斧で原木を壊すと、接続されている原木と葉っぱをまとめて破壊
- BFS（幅優先探索）で効率的に隣接ブロックを検索
- 最大64ブロックまで破壊（パフォーマンス考慮）
- 全種類の原木・葉っぱに対応

## 対応するブロック

### 原木
- オーク、トウヒ、シラカバ、ジャングル、アカシア、ダークオーク
- マングローブ、サクラ
- 木のブロック（全種類）
- 樹皮を剥いだ原木・木のブロック

### 葉っぱ
- 全種類の葉っぱ
- アザレア、開花アザレアの葉っぱ

## 使い方

1. 木の斧をメインハンドに持つ
2. 原木を壊す
3. 接続されている原木と葉っぱが自動で破壊される

## インストール

1. `build/libs/LogDestroyer-<日付>.jar` をサーバーの `plugins` フォルダにコピー
2. サーバーを再起動

## ビルド方法

```bash
cd LogDestroyer
./gradlew build
```

生成されたjarファイルは `build/libs/` に保存されます。

## 対応バージョン

- Minecraft 1.21+
- Paper サーバー

## ライセンス

このプラグインは自由に使用・改変できます。
