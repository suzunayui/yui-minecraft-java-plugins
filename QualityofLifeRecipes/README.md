# QualityofLifeRecipes

生活の質を向上させるクラフトレシピを追加するMinecraftプラグインです。

## 追加レシピ

### シュルカーボックス（16色）

羊毛8個＋チェスト（中央）でシュルカーボックスがクラフトできます。

```
W W W
W C W
W W W
```
- W = 羊毛（各色）
- C = チェスト

| 羊毛の色 | シュルカーボックスの色 |
|---------|---------------------|
| 白 | 白 |
| 橙 | 橙 |
| 赤紫色 | 赤紫色 |
| 空色 | 空色 |
| 黄 | 黄 |
| 黄緑 | 黄緑 |
| 桃 | 桃 |
| 灰 | 灰 |
| 薄灰 | 薄灰 |
| 青緑 | 青緑 |
| 紫 | 紫 |
| 青 | 青 |
| 茶 | 茶 |
| 緑 | 緑 |
| 赤 | 赤 |
| 黒 | 黒 |

## インストール

1. `build/libs/QualityofLifeRecipes-<日付>.jar` をサーバーの `plugins` フォルダにコピー
2. サーバーを再起動

## ビルド方法

```bash
cd QualityofLifeRecipes
./gradlew build
```

生成されたjarファイルは `build/libs/` に保存されます。

## 対応バージョン

- Minecraft 1.21+
- Paper サーバー

## ライセンス

このプラグインは自由に使用・改変できます。
