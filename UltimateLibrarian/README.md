# UltimateLibrarian

交易所を作るのに疲れた人向けのショップNPCプラグインです。

## 機能

- 4種類の司書NPCを召喚
- 各NPCが特定のエンチャント本をエメラルド64個で販売
- NPCは動かない・無敵・右クリックで取引可能

## NPC一覧

| NPC | 販売内容 |
|-----|---------|
| 📚 道具司書 | 修繕、効率V、耐久III、幸運III、シルクタッチ |
| ⚔️ 武器司書 | ダメージ増加V、火属性II、ドロップ増加III、ノックバックII、アンデッド特効V、虫殺しV |
| 🛡️ 防具司書 | ダメージ軽減IV、火炎耐性IV、落下耐性IV、爆発耐性IV、飛び道具耐性IV、棘の鎧III、水中歩行III、水中採掘、水中呼吸III |
| 🏹 遠距離司書 | パワーV、フレイム、無限、パンチII、クイックチャージIII、マルチショット、貫通IV、宝釣りIII、誘惑III |

## コマンド

| コマンド | 説明 |
|---------|------|
| `/ul tool` | 📚 道具司書を召喚 |
| `/ul weapon` | ⚔️ 武器司書を召喚 |
| `/ul armor` | 🛡️ 防具司書を召喚 |
| `/ul ranged` | 🏹 遠距離司書を召喚 |
| `/ul remove` | 全ての司書NPCを削除 |
| `/ul` | ヘルプを表示 |

## インストール

1. `build/libs/UltimateLibrarian-<日付>.jar` をサーバーの `plugins` フォルダにコピー
2. サーバーを再起動

## ビルド方法

```bash
cd UltimateLibrarian
./gradlew build
```

生成されたjarファイルは `build/libs/` に保存されます。

## 対応バージョン

- Minecraft 1.21+
- Paper サーバー

## ライセンス

このプラグインは自由に使用・改変できます。
