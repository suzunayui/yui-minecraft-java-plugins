# サーバー導入プラグインガイド

このサーバーでは、プレイを便利にする以下のプラグインが導入されています。

## プラグイン一覧

| プラグイン | 説明 |
|-----------|------|
| CompactFarms | 自動で資源を生成するシュルカーボックス |
| ContainerSearch | 近くのコンテナの中身を検索 |
| LogDestroyer | 木を一括破壊 |
| Memo | メモを座標付きで保存・検索 |
| PositionHUD | 画面に座標を常時表示 |
| ReturnHome | ホームにテレポート |
| UltimateLibrarian | 交易所NPC |

---

## CompactFarms

シュルカーボックスに自動的に資源を生成するアイテムです。

### クラフトレシピ

| アイテム | レシピ | 生成資源 |
|---------|--------|---------|
| 鉄CompactFarms | 鉄インゴット8個 + チェスト1個 | 鉄インゴット（1分ごと） |
| エメラルドCompactFarms | エメラルド8個 + チェスト1個 | エメラルド（1分ごと） |
| 火薬CompactFarms | 火薬8個 + チェスト1個 | 火薬（1分ごと） |

### 使い方

1. シュルカーボックスを地面に設置する
2. 自動的に資源が生成されます（上限：シュルカーボックス満杯まで）

---

## ContainerSearch

近くのコンテナの中身を検索できます。

### コマンド

```
/cs <アイテム名>
```

### 使用例

```
/cs ダイヤモンド
```

### 検索範囲

- X軸: ±10ブロック
- Z軸: ±10ブロック
- Y軸: -1〜+5ブロック

チェスト、タル、シュルカーボックスなどすべてのコンテナに対応しています。シュルカーボックス内のアイテムも再帰的に検索されます。

---

## LogDestroyer

木の斧で木を一括破壊できます。

### 使い方

1. **木の斧**をメインハンドに持つ
2. 原木を壊す
3. 接続されている原木と葉っぱが自動で破壊される（最大64ブロック）

全種類の原木・葉っぱに対応しています。

---

## Memo

メモを現在地と一緒に保存・検索できます。

### コマンド

| コマンド | 説明 |
|---------|------|
| `/memo <文字列>` | メモを現在地と一緒に保存 |
| `/ms <検索文字列>` | メモを文字列で検索 |
| `/mn` | 近くのメモを5件まで表示 |
| `/md <メモID>` | メモを削除 |
| `/mp <プレイヤー名>` | 指定プレイヤーの直近メモを表示 |

### 使用例

```
/memo ここにダイヤが大量にある
/ms ダイヤ
/mn
/md 1
/mp Yui
```

---

## PositionHUD

画面下部に座標を常時表示します。

### コマンド

| コマンド | 説明 |
|---------|------|
| `/poshud` | 座標表示のON/OFFを切り替え |
| `/poshud <プレイヤー名>` | 特定のプレイヤーの座標を表示 |

### 表示例

```
X: 123  Y: 64  Z: -456
```

デフォルトでONになっています。

---

## ReturnHome

ベッドまたは初期スポーンにテレポートできます。

### コマンド

```
/home
```

### 仕様

- コマンド実行後、10秒間のカウントダウンが発生します
- カウントダウン中に移動するとキャンセルされます
- クールタイム: 10分

---

## UltimateLibrarian

エンチャント本を販売するNPCを召喚できます。

### コマンド

| コマンド | 説明 |
|---------|------|
| `/ul tool` | 道具司書を召喚 |
| `/ul weapon` | 武器司書を召喚 |
| `/ul armor` | 防具司書を召喚 |
| `/ul ranged` | 遠距離司書を召喚 |
| `/ul remove` | 全ての司書NPCを削除 |

### NPC一覧

#### 道具司書
修繕、効率V、耐久III、幸運III、シルクタッチ

#### 武器司書
ダメージ増加V、火属性II、ドロップ増加III、ノックバックII、アンデッド特効V、虫殺しV

#### 防具司書
ダメージ軽減IV、火炎耐性IV、落下耐性IV、爆発耐性IV、飛び道具耐性IV、棘の鎧III、水中歩行III、水中採掘、水中呼吸III

#### 遠距離司書
パワーV、フレイム、無限、パンチII、クイックチャージIII、マルチショット、貫通IV、宝釣りIII、誘惑III

### 価格

すべてのエンチャント本は**エメラルド64個**で販売しています。
