# RecycleGacha

アイテムをリサイクルしてレアアイテムを入手するガチャプラグインです。

## 機能

- チェストの上にホッパーを置き、釣り竿を持ってホッパーを殴るとガチャ機として登録
- ホッパーにアイテムを入れるとポイントが貯まる
- ポイントが一定量貯まると抽選が行われ、下のチェストからレアアイテムが出る
- 量産可能なアイテム（丸石、石、土、原木など）はポイント0

## 使い方

1. チェストを設置
2. チェストの上にホッパーを設置
3. 釣り竿をメインハンドに持ってホッパーを殴る（右クリック）
4. 「ガチャ機を登録しました！」と表示されたら成功
5. ホッパーにアイテムを入れる
6. ポイントが100貯まると抽選が行われる
7. 抽選結果は本人にのみ表示される
8. 大当たりを引いた場合はサーバー全員に通知される

## ポイント設定

| アイテム | ポイント |
|---------|---------|
| 丸石、石、土、原木、板材、鉄インゴット、エメラルド | 0 |
| 石炭、レッドストーン、ラピス、クォーツ、金インゴット | 1 |
| ダイヤモンド、金のリンゴ、経験値瓶 | 5 |
| ネザライトインゴット、エンチャント金のリンゴ | 10 |

## 抽選結果

### 🎉 大当たり（0.1～1%）
- エメラルドブロック 1個
- ダイヤモンド 2個
- エンチャント本（ランダム）

### 💚 当たり（1～5%）
- エメラルド 2個
- ダイヤモンド 1個
- 金インゴット 2個
- 鉄インゴット 2個
- 花火 2個
- 金のリンゴ 1個

### 😊 普通（10～30%）
- パン 1個
- ステーキ 1個
- 焼き鳥 1個
- 矢 1個
- 松明 1個
- 経験値瓶 1個
- エンダーパール 1個

### 😅 ハズレだけど使える（約70%）
- 骨粉 1個
- 糸 1個
- 革 1個
- 石炭 1個
- レッドストーン 1個
- ラピスラズリ 1個

## コマンド

| コマンド | 説明 |
|---------|------|
| `/rg reload` | 設定を再読み込み |

## インストール

1. `build/libs/RecycleGacha-<日付>.jar` をサーバーの `plugins` フォルダにコピー
2. サーバーを再起動

## ビルド方法

```bash
cd RecycleGacha
./gradlew build
```

生成されたjarファイルは `build/libs/` に保存されます。

## 対応バージョン

- Minecraft 1.21+
- Paper サーバー

このプラグインは自由に使用・改変できます。
